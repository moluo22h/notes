# 虚拟环境的安装与配置

## 1.安装python

```bash
sudo apt-get install python
```

## 2.安装virtualenv

```bash
pip install virtualenv
```

## 3.创建虚拟环境

```bash
mkvirtualenv {虚拟环境名称}
```

> 指定python版本创建虚拟环境
>
> ```bash
> mkvirtualenv --python={python执行文件路径} {虚拟环境名称}
> ```
>
> 示例：
>
> ```bash
> mkvirtualenv --python=F:\software_install\Python\Python36\python.exe {虚拟环境名称}
> ```

启动虚拟空间

```
source activate.bat 
```

查看存在的虚拟环境

```bash
workon
```

启动虚拟环境

```bash
workon {虚拟环境名称}
```
