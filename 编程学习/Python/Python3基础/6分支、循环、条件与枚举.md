## 条件语句

方式一：

```python
if condition:
    pass
```

> 注意：condition是一个表达式，可以是bool值，函数

方式二：

```python
if condition:
    pass
else:
    pass
```

方式三：

```python
if condition:
    pass
elif expression:
    pass
else:
    pass
```

> 注意：python中没有switch分支语句，想要实现分支语句请使用elif
