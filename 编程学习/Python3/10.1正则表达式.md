## 正则表达式

正则表达式由**普通字符**(如：python)和**元字符**（如：\d）组成

本节我们来聊聊正则表达式为何强大？

我们认为其强大的因素在于抽象。以下时正则表达式的抽象内容

- 字符集合[123abc$%]

- 概况字符集
- 数量词

## 字符集合

[123abc$%]

[^cfd]

[a-z]

## 概况字符集

```python

\d	数字字符	\D
\w	单词字符	\W
\s	空白字符	\S
.	匹配除换行符\n之外其他所有字符
```

## 数量词

```
{3,6}
*	配置0次或者多次
-	匹配1次或者多次
?	匹配0 次或1次
```

> 当？前为固定字符时，代表数量词；当？前为范围时，代表非贪婪标志符

## 边界匹配

```
^	以什么字符开始
$	以什么字符结尾
```

## 组

组的使用，使得正则表达式拥有了**以字符串为单元**的抽象能力
