# MessageSource

用于解析消息的策略接口，支持此类消息的参数化和国际化。

MessageSource类图如下：

![MessageSource](media/1.5.MessageSource/MessageSource.png)

MessageSource接口要求实现类具备以下能力：

1. 消息参数化
2. 消息国际化
