# selenium环境搭建

搭建selenium环境分为以下三方面：

- client
- driver
- 浏览器

## client

python安装

python IDE安装（pycharm）

seleniun安装

```bash
pip install seleniun
```

> 验证selenium是否安装成功可在python命令行输入
>
> from selenium import webdriver

## driver

下载chromedriver.exe

放置chromedriver.exe到python的安装目录更目录下

## 浏览器

chrome浏览器下载并安装
